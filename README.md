## Instalaci√≥n  

Clone the project  

~~~bash  
  git clone git@github.com:emanuel-ra/rhsys.git
~~~

Go to the project directory  

~~~bash  
  cd rhsys
~~~

Install dependencies  

~~~bash  
npm install
composer install
~~~

Start the server  

~~~bash  
php artisan serve
~~~
